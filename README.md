# Compañía de Vino - Cartas Digitales

Sistema de cartas de vinos digitales con actualización automática desde Google Sheets.

## Cómo usar:
1. Abrir `index.html` en un navegador
2. Las bodegas se cargan automáticamente desde Google Sheets
3. Click en cualquier bodega para ver sus vinos y QR
4. Usar el QR para compartir la carta específica

## Características:
- ✅ Actualización automática de precios
- ✅ QR individual para cada bodega  
- ✅ Navegación completa entre páginas
- ✅ Diseño responsive
